Build a Music Disc mod that behaves like vanilla discs.

Add your .ogg sound files and matching textures (e.g. apple.ogg + apple.png). DiscForge pairs them by name, warns on duplicates, generates a Java Edition mod JAR layout, and downloads it in one click.

How pairing works

Files are matched by base name (case-insensitive). Example: apple.ogg pairs with apple.png. If there are duplicates (same exact filename), you’ll get a warning.

Mod Settings

Note: Browsers can’t truly “compile” Fabric/Forge mods without a build toolchain. This generator creates a jar-shaped zip containing the assets/data scaffolding you’d use in a real mod project (or as a resourcepack-like structure). You can still import these assets into a proper Fabric/Forge template.
